Add the Upstream Remote

Checking the Current Git Remote(s)

To view current remote(s):


            $ git remote
          

            origin

For more verbose output:


            $ git remote -v
          

            origin  git@github.com:username/repo-name.git (fetch)
            origin  git@github.com:username/repo-name.git (push)

Adding Upstream Remote

To add the upstream repo to local remote:


            $ git remote add upstream https://github.com/username/repo-name.git

To confirm the added remote:


            $ git remote
          

            origin
            upstream

Fetch and Merge Upstream Changes

Fetch Upstream Changes

Fetch changes from the upstream repo:


            $ git fetch upstream

Rebase or Merge

Rebase Upstream Changes

Checkout the master branch:


            $ git checkout master
          

            Switched to branch 'master'

Rebase the upstream changes:


            $ git rebase upstream/master
          

            Updating 'a422352..5fdff0f'
            ...

Merge Upstream Changes

Checkout the master branch:


            $ git checkout master
          

            Updating 'a422352..5fdff0f'
            ...

Copying the SSH key to clipboard (Windows)

Copies the contents of the id_rsa.pub file to clipboard: 


            $ clip < ~/.ssh/id_rsa.pub

If clip isn't working, you can locate the hidden .ssh folder, open the file in your favorite text editor, and copy it to your clipboard.

Copying the SSH key to clipboard (Linux)

Download and install xclip (If not install)


            $ sudo pacman -S xclip

Copies the contents of the id_rsa.pub file to clipboard: 


            $ xclip -sel clip < ~/.ssh/id_rsa.pub

If xclip isn't working, you can locate the hidden .ssh folder, open the file in your favorite text editor, and copy it to your clipboard.

Adding a new SSH key to GitHub account

  • Navigate to the GitHub user settings page
  • In the user settings sidebar, click SSH and GPG keys
  • Click New SSH key
  • In the Title field, add a descriptive label for the new key
  • Paste your key into the Key field
  • Click Add SSH key
  • If prompted, confirm your GitHub password

Done!
