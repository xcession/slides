Installation

Created by Reverie Visuals and contributors

Prerequisites

  • Bootable media
  • Internet connection (Ethernet)
  • Boot into the live environment

Preparing

Verifying internet connection

Verify connection with the ping command:


						# ping -c 3 archlinux.org
					

						> PING archlinux.org (138.201.81.199): 56 data bytes
						> 64 bytes from 138.201.81.199: icmp_seq=0 ttl=47 time=224.669 ms
						> 64 bytes from 138.201.81.199: icmp_seq=1 ttl=47 time=226.400 ms
						> 64 bytes from 138.201.81.199: icmp_seq=2 ttl=47 time=224.150 ms
						>
						> --- archlinux.org ping
						statistics ---
						> 3 packets transmitted, 3 packets received, 0.0% packet loss
						> round-trip min/avg/max/stddev = 224.150/225.073/226.400/0.962 ms

Set keyboard layout

List available keyboard layout:


						# ls -R /usr/share/kbd/keymaps

Set keyboard layout:


						# loadkeys uk

(Default console keymap is us)

Verifying the boot mode

To verify the UEFI:


						# ls /sys/firmware/efi/efivars

Updating system clock

To ensure the system clock is accurate:


						# timedatectl set-ntp true

Partition the disk

Listing all available devices (fdisk)

To list all available devices (using fdisk):


							# fdisk -l

Listing all available devices (lsblk)

To list all available devices (using lsblk):


							# lsblk
						

							> NAME  MAJ:MIN RM   SIZE RO TYPE MOUNTPOINT
							> loop0   7:0    0 549.2M  1 loop /run/archiso/sfs/airootfs
							> sda     8:0    0    16G  0 disk
							> L sda1 8:1    0   300M  0 part
							> sr0    11:0    1   671M  0 rom  /run/archiso/bootmnt

Creating a partition table


							# fdisk /dev/sda

Creating a new DOS partition table: (for Legacy systems) 


							Command (m for help): o [Enter]

Creating a new GPT partition table: (for UEFI systems) 


							Command (m for help): g [Enter]

Saving the partition table

Write the partition table to disk and exit:


							Command (m for help): w [Enter]
